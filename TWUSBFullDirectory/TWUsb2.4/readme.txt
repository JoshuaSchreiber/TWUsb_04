Folgende Dateien sind im TWUsb Version 2.4Archiv:

install.html 		Die Installationsanweisung für TWUsb

twusb.jpg 			Ein Bild vom Demo Programm zum testen der Schnittstelle

start_K8055_demo.bat 	Startet eine Java GUI zum testen der Schnittstelle

start_batch.bat 	Startet einen kleinen Testlauf als Konsolenanwendung, der Java Code liegt im src_sample Verzeichnis

TWUsb.dll 		Die C++ DLL Version 24 mit der Java Schnittstelle zu dem USB-Board in das Windows/System32 Verzeichnis kopieren
TWUsb.jar 		Die Java Bibliothek für das USB-Board

log4j-1.2.13.jar Die Lib für das Logging

src_sample 		Dies Verzeichnis enthält Java Beispielcode
api  	 		Dieses Verzeichnis enthält die Dokumentation zur Schnittstelle, die index.html Datei starten und auf Schnittstelle und Description klicken

license.txt 		Die Benutzungsbedingungen der Software
readme.txt 		Diese Datei


release_notes.txt 	Alle Programmänderungen.

/ant/ant-twusb1.1.1.zip Das Archiv mit dem Optionalen TWUsb Ant Task. Für Ant 1.7. Das *.zip Archiv auspacken.
/ant/ant-twusb1.1.1.zip/doc Die Beschreibung des TWUsb Ant Task auf index.html klicken.
/ant/ant-twusb1.1.1.zip/build.xml Ein Beispiel eines Ant Tasks
/ant/ant-twusb1.1.1.zip/ant-twusb.jar In das Ant/ext/lib Verzeichnis kopieren

/server/twusbserver.war Das Web Archiv für die Installation in einem Server
/server/install.html	Die Installations Anleitung für die TWUsbServer Anwendung
/server/twusbserver.jpg Ein Bild der Web-Anwendung
