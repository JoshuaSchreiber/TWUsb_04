TWUsbServer 1.2.0


Dieses Dokument beschreibt die Installation der TWUsbServers Anwendung auf einen Tomcat Server.

Getestet wurde auf Apache Tomcat/6.0.10/Windows XP mit MSIE 6.0 SP2.
Bei andere Versionen und Betriebssysteme muss ähnlich vorgegangen werden.

Installation:

1. Treiber von der Firma Velleman installieren (Version 2.0).
2. TWUsb.dll in das Windows/System32 Verzeichnis kopieren.
3. Tomcat Server 6.0 und JAVA Version >= 1.5 installieren
4. Kopieren des twusbserver.war in das /webapps Verzeichnis von Tomcat oder das twusbserver.war über die Manager Konsole hochladen http://localhost:8080/manager/html
5. Starten des Tomcat Servers durch ausführen /bin/tomcat6.exe bzw. über das Start Menü.
6. Aufrufen der Application im Browser über die URL: http://localhost:8080/twusbserver
7. Die Anwendung erscheint im Browser:


Tips:

Sollte nach dem klicken auf senden eine:
java.lang.UnsatisfiedLinkError: no TWUsb in java.library.path
Meldung kommen, dann findet der Tomcat die TWUsb.dll nicht. Die muss in das Windows/System32 Verzeichnis kopiert werden. Dann muss der Server neu gestartet werden.

Der Port der in der default Einstellung 8080 ist, kann in der /conf/server.xml von Tomcat geändert werden.

Wenn einige Button nicht funktionieren, überprüfen ob JavaScript im Browser angeschaltet ist.
