TWUsb Java Schnittstelle zum "USB Experiment Interface Board K8055" der Firma Velleman.

Das TWUsb ist eine JAVA Schnittstelle für das "USB Experiment Interface Board K8055" der Firma Velleman. Das Programmierinterface für Java besteht aus einer von mir in C++ geschriebenen dynamischen Bibliothek, die twusb.dll und einem von mir entwickelten Java Interface, die twusb.jar Schnittstelle. So kann per JAVA auf alle Funktionen des Board K8055 zugegriffen werden.

Die Installation der TWUsb Schnittstelle.

Quickinstall:

1. Das TWUsb.zip Archiv in ein leeres Verzeichnis auspacken.
2. Die TWUsb.dll in das Windows/System32 Verzeichnis kopieren. Die nötigen Treiber der Firma Vellman müssen installiert sein. D.h. es müssen im Systemverzeichnis die K8055D.dll usw. vorhanden sein und zwar in der Version 2.0.0. Diesen Treiber gibt es auf der Velleman Homepage.
3. Die start_K8055_demo.bat mit einen doppelklick starten.
Es muss nach einigen Sekunden diese Programmfenster erscheinen:



Wenn kein Programmfenster nach dem start_K8055_demo.bat erscheint, die JAVA Version testen, sie sollte > 1.5 sein.


Da das TWUsb Programm ein JAVA Programm ist läuft es Plattformunabhängig auf allen Windows Versionen wie z.b. 98SE, ME, XP, 2000, Vista aber nicht unter Linux, Mac, Unix usw. da die twusb.dll nur für das Betriebssystemen Windows entwickelt wurde.

Download und weitere Infos: http://www.wenzlaff.de/twusb.html

Bisher validierte Betriebssysteme: Windows XP
